HOW TO RUN:

	To run this project you can either run it through the same file simply by
going into command line or bash and using:
		
		python "FILENAME".py


making sure you are in the same directory as the python file.
What is expected as output would be a .csv file with the respective python file name
if the .csv file exists, it would append to same file.

If everything works correctly, there would display a heatgrid with a growing single source as well as a .csv file
